# Projeto Clone Moodle (LMS)

## 1. Introdução
Este projeto visa o desenvolvimento de um Sistema de Gestão de Aprendizagem (LMS) moderno, inspirado nas funcionalidades principais do Moodle, mas focado em usabilidade e performance.

## 2. Objetivos
* Prover uma plataforma para cursos online e treinamentos corporativos.
* Facilitar a gestão de materiais didáticos e avaliações.
* Permitir o acompanhamento de progresso de alunos e emissão de certificados.

## 3. Público-Alvo
* Empresas de treinamento (ex: Segurança do Trabalho/NRs).
* Instituições de ensino e professores autônomos.
* Gestores de RH para treinamento interno.
