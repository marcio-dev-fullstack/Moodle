Especificação Técnica

Projeto Clone Moodle LMS

Plataforma centralizada para cursos online e treinamentos corporativos. Focada em usabilidade, performance e acompanhamento detalhado de progresso.

1. Perfis e Público-Alvo

Esta secção define quem utilizará o sistema e os seus respetivos papéis e permissões dentro da plataforma modular. Clique ou explore os cartões para compreender o foco de cada utilizador.

💻

Administrador

Gestão global do sistema, criação de categorias de cursos e controlo total de utilizadores (banir, promover, editar perfis).

Foco: Gestores de RH, TI Interno
🎓

Professor / Instrutor

Gestão de conteúdos didáticos, criação de cursos, acompanhamento de notas, estatísticas e monitorização do desempenho de alunos.

Foco: Escolas, Autônomos
👤

Aluno

Inscrição em cursos, consumo de vídeos e materiais em PDF, realização de questionários e obtenção automática de certificados.

Foco: Colaboradores, Estudantes

3. Modelo de Dados e Arquitetura

Explore as entidades principais do sistema e os seus atributos cruciais para a base de dados relacional. Passe o rato por cima (hover) sobre cada tabela para visualizar os seus campos.

User

Tabela

Utilizadores do sistema.

  • id (PK)
  • nome
  • email
  • password_hash
  • role
  • data_registo

Course

Tabela

Cursos ofertados.

  • id (PK)
  • titulo
  • descricao
  • capa_url
  • professor_id (FK)
  • categoria_id (FK)

Module

Tabela

Módulos dos cursos.

  • id (PK)
  • curso_id (FK)
  • titulo
  • ordem

Lesson

Tabela

Aulas e conteúdos.

  • id (PK)
  • modulo_id (FK)
  • titulo
  • conteudo
  • tipo
  • url_media
  • ordem

Enrollment

Tabela

Vínculo aluno-curso.

  • id (PK)
  • aluno_id (FK)
  • curso_id (FK)
  • progresso_percentual
  • data_inicio
